# Rule-VLN Project Page

This directory contains the static project website for:

**Rule-VLN: Bridging Perception and Compliance via Semantic Reasoning and Geometric Rectification**

The page is designed in the common CV conference project-page style and includes prominent links for the paper, code, and dataset.

## Local Preview

From this directory:

```bash
python -m http.server 8126 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8126/
```
